     🚀 STEP-BY-STEP INSTRUCTIONS:
    ══════════════════════════════════════════════════════════════

      STEP 1: Create GitHub Repository
      ─────────────────────────────────
         1. Go to: https://github.com/new
         2. Repository name: fallguard-app
         3. Make it PUBLIC
         4. Click "Create repository"
         5. Copy the repository URL


      STEP 2: Push Your Code
      ───────────────────────
         Option A - Use Batch File (EASIEST):
            Double-click: PUSH_TO_GITHUB.bat
            Enter your repository URL when asked

         Option B - Manual Commands:
            cd C:\Users\Sameer\Downloads\50hz\50hz\mobile_app
            git init
            git add .
            git commit -m "Initial commit"
            git remote add origin YOUR_REPO_URL
            git push -u origin main


      STEP 3: Watch the Build
      ────────────────────────
         1. Go to your repository on GitHub
         2. Click "Actions" tab
         3. You'll see "Build Android APK" running
         4. Wait 30-40 minutes


      STEP 4: Download APK
      ─────────────────────
         1. When build completes (green checkmark)
         2. Click on the workflow run
         3. Scroll down to "Artifacts" section
         4. Click "FallGuard-APK" to download ZIP
         5. Extract ZIP to get APK file


      STEP 5: Install & Configure
      ────────────────────────────
         1. Transfer APK to Android phone
         2. Install (enable Unknown sources)
         3. Open app → Settings
         4. Emergency Contact: +923001234567
         5. Server URL: https://web-production-2755d.up.railway.app
         6. Save → Test SMS → Test Connection
